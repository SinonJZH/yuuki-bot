## 指令表
-----通用-----  
/help (指令)： 显示这个页面，或指定指令的帮助  
-----骰子-----  
/r [掷骰表达式] ：投掷骰子，支持加法，支持惩罚/奖励骰  
/check [检定数值] (鉴定属性) ：对某一属性进行检定，支持惩罚/奖励骰
